# 프로젝트 소개
Melting은 인기 커뮤니티 사이트를 통합한 커뮤니티 웹 사이트입니다.<br>
디시인사이드, 더쿠 등 다양한 인기 사이트를 실시간으로 제공합니다.


# 기능
- 10분 단위로 유머 사이트 카테고리별 최신글 목록 확인
- 회원가입, 로그인
- 게시글 쓰기, 수정, 삭제, 추천 이미지 다운로드
- 게시글 검색
- 댓글 쓰기, 삭제
- 회원정보 수정, 작성한 글 확인, 남긴 댓글 확인
